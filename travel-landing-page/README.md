Travel Website Landing Page (React UI)

This project is a responsive travel website landing page built using React by referencing the provided Figma design. The goal of this assignment was to convert a UI design into reusable React components while maintaining clean layout structure, consistent styling, and a smooth user experience.

The project focuses purely on frontend implementation and does not include any backend or API integrations.

Tech Stack

1. React (Functional Components)
2. Tailwind CSS
3. Lucide React Icons
4. Vite (for development setup)

Sections Implemented

The landing page is divided into the following sections: 1. Hero section with call-to-action 2. Category section to explore travel types 3. Popular destinations section with cards 4. Why Choose Us section highlighting key features 5. Testimonials section for user feedback 6. Subscribe section

Each section is implemented as a separate React component to keep the code modular and easy to maintain.

Design Reference:
The UI layout and section flow are based on the Figma design shared as part of the assignment. While the overall structure follows the provided design, minor adjustments were made to colors, spacing, and text to improve readability and visual balance.
